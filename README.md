# 📊 Stack Overflow 2025 개발자 설문조사 분석 프로젝트

Streamlit을 활용한 인터랙티브 데이터 분석 대시보드

---

## 1. 프로젝트 설명

이 프로젝트는 전 세계 개발자들을 대상으로 한 **Stack Overflow의 2025년 최신 설문조사 데이터**를 분석하고 시각화하는 인터랙티브 웹 대시보드입니다. 

사용자는 이 대시보드를 통해 국가별, 기술별로 최신 개발자 트렌드를 쉽게 탐색하고 비교해 볼 수 있습니다.

## 2. 주제

**Stack Overflow 2025 개발자 설문조사 데이터 분석**

- **데이터 소스:** Stack Overflow Developer Survey 2025 (Public CSV)

## 3. 주제 선택 이유

프로그래밍 언어의 인기 순위나 개발자 트렌드는 IT 분야에 종사하거나 공부하는 학생들에게 항상 가장 큰 관심사 중 하나입니다. 

단순히 정적인 리포트를 보는 것을 넘어, **사용자가 직접 데이터를 탐색**하며 궁금한 점(예: '우리나라에서는 어떤 언어를 가장 배우고 싶어할까?')을 즉각적으로 확인할 수 있는 **인터랙티브 대시보드**를 구현해보고자 이 주제를 선택하게 되었습니다.

## 4. 데이터 분석 내용

Streamlit을 기반으로 3개의 주요 메뉴를 가진 대시보드를 구현했습니다.

### 📄 1. HOME
- 프로젝트의 개요와 로드된 원본 데이터를 간략하게 보여줍니다.

### 🚀 2. 언어 사용 현황
- **국가별 필터 기능 (Sidebar)**: 사이드바에서 'All' 또는 특정 국가(예: South Korea)를 선택할 수 있습니다.
- **Top 15 시각화**: 선택된 국가에서 **'가장 많이 사용한 언어'** Top 15를 집계하여 Plotly 바 차트로 시각화합니다.

### 🔍 3. 언어 심층 분석
- **동적 분석 기능**: 사용자가 '가장 많이 사용한 언어'뿐만 아니라, 다음과 같은 다양한 주제를 직접 선택하여 분석할 수 있습니다.
    - 가장 배우고 싶은 언어 (`LanguageWantToWorkWith`)
    - 가장 많이 사용한 데이터베이스 (`DatabaseHaveWorkedWith`)
    - 가장 배우고 싶은 데이터베이스 (`DatabaseWantToWorkWith`)
    - 가장 많이 사용한 플랫폼 (`PlatformHaveWorkedWith`)
- **데이터 전처리**: 'Python;SQL'처럼 세미콜론(;)으로 묶인 응답 데이터를 `split`과 `explode` 함수를 이용해 파싱하고 집계하는 로직을 구현했습니다.
